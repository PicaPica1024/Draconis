生态学外业探险游戏需求文档
# 1. 游戏名称
野外采样模拟器

# 2. 游戏描述
一款以生态考察为主题的回合制文字冒险游戏,玩家扮演生态学研究生,在有限的时间和预算内完成采样任务,通过合理决策平衡各项状态值,达成项目目标。

# 3. 玩法说明
## 3.1 游戏目标
“在截止日期之前，不超预算的保质保量完成采样任务，并尽可能腾出时间来玩耍”
即：
在time left >= 0时,同时满足以下条件即游戏胜利:
budget > 0
data quality > 80
sample = 100
trust > 50
或
time left >= 0时满足
dragon =2

游戏失败条件:
任意回合结束时，满足任一项游戏立即失败
time left = 0
budget <= 0
trust = 0
data quality <50

## 3.2 玩家状态值
*time left*
距离项目截至日期剩余天数,起始值为20。每一个回合-1

*energy*
团队士气,区间0-10,小于2时无法开展任何操作除了休息

*budget*
团队预算,起始值10000,处理事件需要消耗预算,請款、賺外快可以增加预算.每一个回合-500（交通和食宿的固定消耗）

*healthy*
健康值,区间0-1,起始值为1.健康值为0时无法开展任何工作，除了休息。

*data quality*
数据质量,区间0-100,起始值100，低于50时游戏失败。

*sample*
数据量,区间0-100,起始值0,成功采样并成功返回可以增加数据量,一些意外事件会降低数据量,例如忘了备份、记录本损坏、记录本遗失等。

*trust*
信任值,起始值100,任何应对意外事件的請款行为都可能降低信任值,采样中捅了篓子也会降低信任值,被媒体采访和增加团队凝聚力可以增加信任值,信任值小于50时将无法請款、无法申请延长时间,信任值为0时游戏直接失败。

*slack*
摸鱼时间,起始值为0，为触发游戏胜利条件时的time left值,摸鱼时间越多成就越高,当游戏结束时显示摸鱼时间。

*instrument*
仪器，起始值为100，采样过程中有可能损坏仪器，降低仪器值。当仪器值小于50时，无法开展采样，采购物资可以提高仪器值。

*dragon*
龙，彩蛋剧情。第一次触发到dragon +1的结局时，显示该状态。当dragon =2时，游戏直接获胜。

## 3.3 回合机制
游戏使用回合制,每一个回合执行以下流程:

每个回合执行：
time left = time left - 1
budget = budget -500

并在以下六个动作中的选择一个执行:
C采样
F复核数据
X休息
Q请款
S申请延期
W采购与维修

# 4. 剧情系统
每一个回合，玩家选择“动作“后会触发随机事件。玩家可以自主选择如何应对随机事件。每一种选择，都会产生不同的效果，并反映在玩家状态值的变化上。

**结局 C3AA01 (50%)**
